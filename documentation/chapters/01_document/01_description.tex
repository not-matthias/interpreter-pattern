\subsection{Abstract Syntax Tree}

]subsubsection{}

For example you have a simple JavaScript function: 
\begin{verbatim}
function foo(x) {
if (x > 10) {
    var a = 2;
    return a * x;
}

return x + 10;
}
\end{verbatim}
